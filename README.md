# 🎮 Hangman Game

This is a simple Hangman game built using Python.

## 📌 Features
- Random word selection
- User input for guessing letters
- Limited attempts
- Win/Loss result

## ▶️ How to Run

1. Open Command Prompt
2. Go to folder:
   cd Desktop
3. Run:
   python hangman.py

## 🧠 Concepts Used
- Loops
- Conditions
- Lists
- Random module
